# MSA Decision Guide

## Study Selection

| Situation | Study |
| --- | --- |
| Same parts measured by each operator multiple times | Crossed Gage R&R |
| Parts cannot be remeasured by every operator | Nested Gage R&R |
| Known reference values across measurement range | Bias and linearity |
| Same master or sample measured over time | Stability |
| Pass/fail or ordinal inspection | Attribute agreement |

## Continuous Gage R&R Checks

- Confirm parts span the process variation or tolerance range.
- Confirm operators represent normal production appraisers.
- Confirm repeated trials are randomized or blinded where feasible.
- Check part-by-operator interaction before simplifying.
- Interpret percent GRR against intended use, not as a universal pass/fail.
- Report ndc and whether the system can distinguish meaningful process categories.

## Attribute Agreement Checks

- Use known standards when available.
- Separate repeatability within appraiser from reproducibility between appraisers.
- Report false accepts and false rejects when the standard is known.
- Use kappa carefully when prevalence is extreme.

## Interpretation

If MSA is weak, recommend measurement improvement before strong claims about stability, capability, DOE effects, or supplier performance.
